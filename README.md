মেসেজের ভেরিয়েবলগুলি ব্যবহার করো:

- [Sender Name] → যিনি মেহেদি চাইছে
- [Receiver Name] → যাকে মেহেদি চাইছে
- [Relation] → Sender এবং Receiver এর সম্পর্ক (Crush, Boyfriend, Brother, Best Friend, Husband, ইত্যাদি)

মেসেজটি হতে হবে **সংক্ষিপ্ত (১–৪০ শব্দ)**, মজার, এবং viral-shareable vibe রাখতে হবে।

উদাহরণ:

- "[Receiver Name], ঈদের দিন আমার হাতে মেহেদি না থাকলে আমার ঈদ incomplete 😢 প্লিজ, আমার জন্য মেহেদি কিনে দাও 😏"
- "[Receiver Name], তুমি না দিলে আমি ৩ দিন কথা বলব না 😈 তাই এখনই লিংক খুলে মেহেদি কিনে দাও 💅"
